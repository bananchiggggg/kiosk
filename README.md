
# Платформа Реестра Транспорта (Киоск)

PWA для планшета, осуществляющее поиск по Google Таблице.

## Правила валидации госномеров

Приложение применяет строгую фильтрацию вводимых данных для уменьшения количества ложных запросов:

1.  **Нормализация**: Все вводы автоматически переводятся в верхний регистр, удаляются пробелы, дефисы и точки. Похожие кириллические буквы (А, В, Е и др.) заменяются на латинские аналоги для поиска.
2.  **Запрет телефонов**: Номера, похожие на телефонные (начинающиеся на +7, 7, 8 или состоящие только из 10-12 цифр), отклоняются.
3.  **Валидные форматы**:
    *   **РФ (Строго)**: Формат `А123ВС77` или `А123ВС177` (Буква, 3 цифры, 2 буквы, 2-3 цифры региона).
    *   **СНГ (Расширенный)**: Любая комбинация букв и цифр длиной от 6 до 10 символов, содержащая хотя бы одну букву и одну цифру.
4.  **Обработка ошибок**: При вводе некорректного значения кнопка поиска блокируется, и выводится подсказка на русском языке.

## Тест-кейсы
*   ✅ **Корректные**: `А123ВС77`, `A123BC77`, `А 123 ВС 77`, `А-123-ВС-77`.
*   ❌ **Некорректные**: `+79991234567`, `89991234567`, `123456` (нет букв), `abcdef` (нет цифр), пустая строка.

## Конфигурация Backend (Google Apps Script)

1. Убедитесь, что у вас есть доступ к таблице: `1-ra-7n7R9siWg7SCQiDq7-lRPwZ8JhskLc0WXy7VpaE`.
2. В таблице должен быть лист с данными (по умолчанию первый).
3. Перейдите в `Расширения` > `Apps Script`.
4. Вставьте код из файла `backend/Code.gs`.
5. Перейдите в `Настройки проекта` (иконка шестеренки).
6. В разделе `Свойства скрипта` добавьте свойство `API_KEY`.
7. Выполните развертывание как Веб-приложение (Execute as: Me, Access: Anyone).
